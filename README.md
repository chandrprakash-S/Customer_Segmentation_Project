# Customer Segmentation using Unsupervised Learning

## Project Description
This project uses K-Means Clustering to divide customers into different groups based on annual income and spending score.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Algorithm Used
- K-Means Clustering

## Features
- Data Visualization
- Elbow Method
- Customer Clustering
- Cluster Visualization

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the project:

python main.py
